# gopray
a prayer helper written in go for Orthodox Christians and all followers of Christ.

## installation
go get github.com/hajimehoshi/oto
go get github.com/hajimehoshi/go-mp3

##usage and flags
./pray (defaults to the trisagion)
./pray --prayer=morning (a morning prayer)
./pray --prayer=night (prayer before bed)
./pray --prayer=creed (Christian creed)
